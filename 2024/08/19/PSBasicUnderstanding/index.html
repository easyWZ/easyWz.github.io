PS（一）基本操作

基本认识

  1. 打开ps - 主页 - 新建
  2. 整个ps分区域显示
    1. 最顶层-菜单栏
    2. 左侧-工具栏
    3. 顶部第二层-属性栏（父级工具栏）
    4. 右侧-浮动面板（可以长按拖动 随意改变位置）
      1. 打开：菜单 - 窗口 - xx
      2. 重置：菜单 - 窗口 - 工作区 - 复位基本功能
  3. 画布分层
    1. 顶部 - 标题栏
    2. 中间 - 画布
    3. 左下 - 状态栏 更改% 像素
  4. 预设（快捷键）
    1. ctl+ K （mac为 command + k）打开首选项
    2. 首选项 - 文件处理 - 自动存储恢复信息的间隔 - 5分钟
    3. 界面 - 颜色方案 - 黑色

基础操作（图片基本操作）

  1. 置入图片

    1. 单独处理： 文件 - 打开 - 选择本地图片
    2. 添加至已创建文件：
      1. 拖动图片至操作区域
      2. 文件 - 置入嵌入对象 （嵌入源文件）
      3. 文件 - 置入链接的智能对象 （只插入源文件的链接地址，省内存，但发送别人依据地址就会丢失）
      4. 图片进入后会显示一个叉号 说明是一个智能对象文件，可放大缩小，按回车即可确认最终尺寸，叉号消失

  2. 删除 - 选中点击 delete键

  3. 撤回

    1. 右上方 有个回滚箭头标识 点击 展开历史记录 点击对应步骤删除即可
    2. 菜单栏 - 编辑 - 还原删除状态 （快捷键ctl+z）

  4. 放大和缩小

    1. 放大/缩小 option + 上/下滑
    2. ctl + 加号/减号

  5. 图层

    1. 添加图层
      1. 点击右下角 + 号标识
    2. 改变位置
      1. 单个：选中任意单个图层 + 工具栏选中 - 移动工具 + ctl + t 鼠标拖动即可
      2. 整体：shift选中所有图层拖动 + ctl + t 鼠标拖动即可
    3. 删除图层
      1. 拖动至右下角垃圾桶标识
      2. 选中 - delete键

  6. 存储/导出

    1. 文件 - 存储
    2. 文件 - 存储为
    3. 文件 - 导出
    4. 可点击图层处背景前眼睛 关闭背景 导出为透明图

快捷键

从对话框新建一个图层 【Ctrl】+【Shift】+【N】
以默认选项建立一个新的图层 【Ctrl】+【Alt】+【Shift】+【N】
通过拷贝建立一个图层 【Ctrl】+【J】
通过剪切建立一个图层 【Ctrl】+【Shift】+【J】
与前一图层编组 【Ctrl】+【G】
取消编组 【Ctrl】+【Shift】+【G】
向下合并或合并联接图层 【Ctrl】+【E】
合并可见图层 【Ctrl】+【Shift】+【E】
盖印或盖印联接图层 【Ctrl】+【Alt】+【E】
盖印可见图层【Ctrl】+【Alt】+【Shift】+【E】
将当前层下移一层 【Ctrl】+【【】
将当前层上移一层 【Ctrl】+【】】
将当前层移到最下面 【Ctrl】+【Shift】+【【】
将当前层移到最上面 【Ctrl】+【Shift】+【】】
激活下一个图层 【Alt】+【【】
激活上一个图层 【Alt】+【】】
激活底部图层 【Shift】+【Alt】+【【】
激活顶部图层 【Shift】+【Alt】+【】】
调整当前图层的透明度(当前工具为无数字参数的,如移动工具) 【0】至【9】
保留当前图层的透明区域(开关) 【/】
投影效果(在”效果”对话框中) 【Ctrl】+【1】
内阴影效果(在”效果”对话框中) 【Ctrl】+【2】
F1 -帮助
F2 -剪切
F3 -拷贝
F4-粘贴
F5-隐藏/显示画笔面板
F6-隐藏/显示颜色面板
F7-隐藏/显示图层面板
F8-隐藏/显示信息面板
F9-隐藏/显示动作面板
F12-恢复
Shift+f5-填充
Shift+f6-羽化
Shift+f7-选择→反选
ctrl+h-隐藏选定区域
ctrl+d-取消选定区域
ctrl+w-关闭文件
ctrl+Q-退出PHOTOSHOP
